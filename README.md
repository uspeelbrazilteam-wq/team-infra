# Engineering Standards — iGEM EEL-USP

Este repositório define os padrões técnicos e organizacionais do time.

## Objetivo

Garantir consistência, escalabilidade e qualidade no desenvolvimento de todos os projetos.

## Escopo

- Versionamento (Git)
- Fluxo de desenvolvimento
- Estrutura de repositórios
- Regras de contribuição

## Princípios

- NENHUM código é enviado diretamente para a branch principal
- Todo desenvolvimento ocorre via Pull Request
- Clareza e rastreabilidade são obrigatórias;
