# Git, Conceitos Básicos

Git é o sistema de versionamento padrão do time.

## Conceitos

- **Commit:** unidade de alteração
- **Branch:** linha de desenvolvimentto
- **Remote:** repositório no GitHub

## Comandos Essenciais

```bash
git init
git add .
git commit -m "feat: descricao"
git push origin feature/nome
```

Uso full definido em `workflow.md`.
